# QR 개인별 안내문 생성기

고교학점제 수강신청 QR 그리드 PDF(학번·이름·QR코드가 표로 인쇄된 형태)를 학생 1인당 1개의 안내문 docx로 변환하는 로컬 도구입니다.

> **주의**: 여기서 다루는 QR코드는 hscredit.net(고교학점제) 개인별 인증 링크입니다. 스마트교무실 본체(Firebase 앱)와는 별개의 외부 시스템 데이터이며, 개인정보이므로 생성된 파일을 외부에 공유하지 마세요.

## 이 폴더가 스마트교무실 저장소에 있는 이유

이 도구는 Python + Streamlit으로 만들어져 있어 React/Firebase로 구성된 본체 앱과는 런타임이 다릅니다. `firebase deploy` 대상이 아니며, **각 담당 선생님 PC에서 로컬로 실행**하는 별도 유틸리티입니다. 관련 도구들을 한 저장소에서 버전 관리하기 위해 `tools/` 아래에 두었습니다.

## 기능

1. QR 그리드 PDF 업로드 → 학번(5자리)·이름·반·번호 자동 인식, 학생별 QR 카드로 크롭
2. 안내문 양식(docx) 업로드 → `{{이름}}`, `{{학번}}`, `{{반}}`, `{{번호}}` 자동 치환 + QR 이미지 삽입
3. 결과물 다운로드
   - 학생별 개별 docx를 담은 zip
   - 인쇄 편의를 위해 전체를 한 파일로 합친 통합 docx (학생 사이 페이지 나눔 포함)

이름 인식에 실패한 셀(원본 PDF 데이터 결측)은 전체를 중단시키지 않고 별도 경고로 표시한 뒤 생성 대상에서 제외합니다.

## 학번 규칙

5자리 학번 기준: 1번째 자리 = 학년, 2~3번째 자리 = 반, 4~5번째 자리 = 번호. (예: `10611` → 1학년 6반 11번)

## 안내문 양식 작성 규칙

- `{{이름}}`, `{{학번}}`, `{{반}}`, `{{번호}}` — 프로그램이 자동으로 채우는 필드. 문구는 자유롭게 수정하되 이 표기 자체는 그대로 둘 것.
- QR 이미지 자리에는 임의의 이미지를 하나 넣어두면, 생성 시 해당 이미지가 실제 QR로 교체됩니다.

## 실행 방법

```bash
cd tools/qr-notice-generator
pip install -r requirements.txt
streamlit run app.py
```

브라우저가 자동으로 열리지 않으면 터미널에 출력되는 `http://localhost:8501`로 접속합니다.

## 파일 구성

| 파일 | 역할 |
|------|------|
| `app.py` | Streamlit 웹 UI |
| `qr_pipeline.py` | PDF 파싱, docx 치환/병합 등 핵심 로직 |
| `requirements.txt` | 필요 패키지 목록 |
