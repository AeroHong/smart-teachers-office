스마트 출결 시스템

시범운영중

교사가 이벤트(수업·조회·행사 등)를 생성하고 QR 코드를 발급하면, 학생이 스마트폰으로 스캔해 자동으로 출석이 기록됩니다. 미출석자는 Firebase Functions로 자동 결석 처리되어 교사의 수동 작업을 최소화합니다.

QR 코드 발급 및 학생 자동 체크인 (별도 앱 설치 불필요)
단일 이벤트 + 매주 반복 이벤트 (정규 수업) 지원
조회 이벤트 지각 기준시간 설정 — 자동 지각 판정

개인 운영중

Google Classroom의 과제를 불러와 교사가 채점 기준을 작성하면, Gemini AI가 학생 답안을 자동 채점하고 결과를 이메일로 발송합니다. 설문지·문서·스프레드시트·프레젠테이션 등 다양한 과제 형식을 지원합니다.

Google Classroom API 연동 — 수업·과제 자동 불러오기
채점 기준 작성 UI + Gemini AI 자동 채점
설문지 / 구글 문서 / 스프레드시트 / 프레젠테이션 지원
채점 결과 학생별 이메일 자동 발송
