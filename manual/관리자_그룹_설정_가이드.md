## 1. 교사 계정 승인

새로 가입한 교사는 **승인 대기** 상태입니다.  
관리자 페이지에서 승인해야 사용 가능합니다.

### 진입 경로

스마트 출결 시스템 → 좌측 메뉴 **관리자**

![C01 — 관리자 메뉴 진입 화면](./image/img-C01.png)

### 승인 처리

![C02 — 승인 대기 탭 — 교사 목록 및 승인/거절 버튼](./image/img-C02.png)

| 버튼 | 설명 |
|------|------|
| **교사 승인** | 일반 교사 권한 부여 |
| **학교관리자 승인** | 관리자 권한 부여 (교무부장 등) |
| **거절** | 가입 거절 |

## 2. 공유 그룹 만들기 (학급 전체 과목용)

국어·수학·영어 등 학급 단위 수업을 담당하는 모든 교사가 공통으로 사용할 학급 그룹입니다.  
**한 번만 만들면 해당 학급 담당 교사 전원이 이벤트에서 선택할 수 있습니다.**

### 2-1. 학생 명단 관리 진입

좌측 메뉴 **학생 명단 관리** → **새 그룹 만들기** 섹션

![C03 — 새 그룹 만들기 섹션 — 관리자 화면 (공유 그룹 체크박스 포함)](./image/img-C03.png)

### 2-2. 설정 항목

| 항목 | 설정값 | 설명 |
|------|--------|------|
| 그룹 이름 | `3학년 7반 2026` | 학년·반·연도 포함 권장 |
| **공유 그룹** | ✅ 체크 | 모든 교사가 선택 가능 |
| 메인 담당교사 | 해당 담임교사 선택 (선택 사항) | 대표 담당교사 표시용 |
| CSV 파일 | 해당 반 전체 학생 파일 | 나이스 학급 명렬표 |

![C04 — 공유 그룹 체크박스 활성화 후 화면 — 메인 담당교사 셀렉터 표시](./image/img-C04.png)

### 2-3. 저장 확인

저장 후 그룹 목록에서 파란 **공유** 배지가 표시되면 정상입니다.

![C05 — 그룹 목록 — '공유' 배지가 붙은 그룹 카드](./image/img-C05.png)

## 3. 개인 그룹 위임 만들기 (선택과목 교사 대리 생성)

선택과목 교사가 직접 그룹을 만들기 어려운 경우, 관리자가 대리 생성하고 해당 교사에게 귀속시킬 수 있습니다.

### 3-1. 설정 항목

| 항목 | 설정값 | 설명 |
|------|--------|------|
| 그룹 이름 | `물리학Ⅱ-A반 2026` | |
| **공유 그룹** | ☐ 체크 해제 | 개인 그룹 |
| 담당 교사 | 해당 교사 선택 | 이 교사만 이벤트에서 선택 가능 |
| CSV 파일 | 해당 분반 학생 파일 | |

![C06 — 공유 체크 해제 + 담당 교사 선택 화면](./image/img-C06.png)

> 담당 교사로 지정된 선생님의 이벤트 생성 화면에서만 이 그룹이 보입니다.

## 4. 그룹 설정 체크리스트

학기 초 배포 전 아래 항목을 순서대로 완료하세요.

```
[ ] 전체 교사 계정 승인 완료
[ ] 1학년 각 반 공유 그룹 생성 (1학년 1반 ~ N반)
[ ] 2학년 각 반 공유 그룹 생성
[ ] 3학년 각 반 공유 그룹 생성
[ ] 선택과목 분반 그룹 생성 (필요 시 담당 교사 귀속)
[ ] 교사들에게 매뉴얼 배포
```

> **Tip**: 공유 그룹 이름에 연도를 포함하면 (`3학년 7반 2026`) 다음 학년도에 헷갈리지 않습니다.
