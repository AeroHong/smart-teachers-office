## 공통 사항

### 학번 5자리 구조

```
예시: 2  01  07
      ↑  ↑↑  ↑↑
      │   │   번호 (2자리)
      │   반 (2자리)
      학년 (1자리)
```

| 예시 학번 | 학년 | 반 | 번호 | 비고 |
|-----------|------|----|------|------|
| `20107` | 2학년 | 1반 | 7번 | |
| `10601` | 1학년 | 6반 | 1번 | |
| `20729` | 2학년 | 7반 | 29번 | |
| `0` 또는 `None` | | | | → _______________ |

### 구획(분반) 알파벳 의미

| 값 | 의미 | 비고 |
|----|------|------|
| `A`, `B`, `C`, `D`, `E` | _______________ | 같은 과목 내 수업 그룹 |
| 분반 숫자 `1`, `2`... | _______________ | 거의 대부분 1 |
